## 🚀 8. Weather Observation Station 2

### 💡 Idea

We need to calculate:

* The **sum** of all values in the `LAT_N` column.
* The **sum** of all values in the `LONG_W` column.

Both results should be rounded to **2 decimal places**.

---

### 💻 Solution

```sql
SELECT
    ROUND(SUM(LAT_N), 2),
    ROUND(SUM(LONG_W), 2)
FROM STATION;
```

---

### 🛠️ How It Works

1. Start with the **STATION** table.
2. Use the `SUM()` function to calculate the total of `LAT_N`.
3. Use the `SUM()` function again to calculate the total of `LONG_W`.
4. Apply `ROUND(..., 2)` to display each value with two decimal places.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(scans all rows in the `STATION` table)*

**Space Complexity:** `O(1)`
