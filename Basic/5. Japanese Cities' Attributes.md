## 🚀 5. Japanese Cities' Attributes

### 💡 Idea

We need to retrieve **all columns** for cities in the `CITY` table where the `COUNTRYCODE` is `'JPN'`.

This can be done by filtering the rows using the `WHERE` clause.

---

### 💻 Solution

```sql
SELECT * FROM CITY
WHERE COUNTRYCODE = 'JPN';
```

---

### 🛠️ How It Works

1. Start with the **CITY** table.
2. Use the `WHERE` clause to filter rows where `COUNTRYCODE` is `'JPN'`.
3. `SELECT *` returns all columns for the matching cities.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(scans all rows in the `CITY` table)*

**Space Complexity:** `O(1)` *(excluding the output result set)*
