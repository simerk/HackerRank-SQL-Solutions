## 🚀 4. Select By ID

### 💡 Idea

We need to retrieve **all columns** for the city whose `ID` is **1661**.

Since `ID` uniquely identifies a city, we can use the `WHERE` clause to filter the required record.

---

### 💻 Solution

```sql
SELECT * FROM CITY
WHERE ID = 1661;
```

---

### 🛠️ How It Works

1. Start with the **CITY** table.
2. Use the `WHERE` clause to filter the row where `ID` equals `1661`.
3. `SELECT *` returns all columns for the matching city.
4. Since `ID` is unique, only one row is returned.

---

### ⏱️ Complexity

**Time Complexity:** `O(1)` *(if `ID` is indexed or a primary key; otherwise `O(n)`)*

**Space Complexity:** `O(1)` *(excluding the output result set)*
