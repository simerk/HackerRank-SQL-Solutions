## 🚀 7. Weather Observation Station 1

### 💡 Idea

We need to display the **city** and **state** for every record in the `STATION` table.

No filtering or sorting is required.

---

### 💻 Solution

```sql
SELECT CITY, STATE
FROM STATION;
```

---

### 🛠️ How It Works

1. Start with the **STATION** table.
2. Select the `CITY` and `STATE` columns.
3. Return every record since there is no `WHERE` clause.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(scans all rows in the `STATION` table)*

**Space Complexity:** `O(1)` *(excluding the output result set)*
