## 🚀 6. Japanese Cities' Names

### 💡 Idea

We need to retrieve only the **NAME** of cities whose `COUNTRYCODE` is `'JPN'`.

The `WHERE` clause filters Japanese cities, while `SELECT NAME` returns only the required column.

---

### 💻 Solution

```sql
SELECT NAME FROM CITY
WHERE COUNTRYCODE = 'JPN';
```

---

### 🛠️ How It Works

1. Start with the **CITY** table.
2. Filter rows where `COUNTRYCODE` equals `'JPN'`.
3. Return only the `NAME` column for each matching city.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(scans all rows in the `CITY` table)*

**Space Complexity:** `O(1)` *(excluding the output result set)*
