## 🚀 1. Revising the Select Query I

### 💡 Idea

We need to retrieve **all columns** for every city in the `CITY` table where the **population is greater than 100000**.

This can be done by filtering the rows using the `WHERE` clause.

---

### 💻 Solution

```sql
SELECT *
FROM CITY
WHERE POPULATION > 100000;
```

---

### 🛠️ How It Works

1. Start with the **CITY** table.
2. Use the `WHERE` clause to filter rows where `POPULATION` is greater than `100000`.
3. `SELECT *` returns all columns for the cities that satisfy the condition.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(scans all rows in the `CITY` table)*

**Space Complexity:** `O(1)` *(excluding the output result set)*
