## 🚀 2. Revising the Select Query II

### 💡 Idea

We need to retrieve the **NAME** of every city in the `CITY` table where the **population is greater than 120000** and the **country code is `'USA'`**.

This can be achieved by filtering the records using multiple conditions in the `WHERE` clause combined with the `AND` operator.

---

### 💻 Solution

```sql
SELECT NAME
FROM CITY
WHERE COUNTRYCODE = 'USA'
  AND POPULATION > 120000;
```

---

### 🛠️ How It Works

1. Start with the **CITY** table.
2. Use the `WHERE` clause to filter cities whose `COUNTRYCODE` is `'USA'`.
3. Apply another condition to keep only cities with a `POPULATION` greater than `120000`.
4. Return only the `NAME` column for the matching cities.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(scans all rows in the `CITY` table)*

**Space Complexity:** `O(1)` *(excluding the output result set)*
