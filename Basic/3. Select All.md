## 🚀 3. Select All

### 💡 Idea

We need to retrieve **all columns** and **all records** from the `CITY` table.

Since no filtering or sorting is required, we can simply use the `SELECT *` statement.

---

### 💻 Solution

```sql
SELECT * FROM CITY;
```

---

### 🛠️ How It Works

1. Start with the **CITY** table.
2. `SELECT *` tells SQL to retrieve every column.
3. Since there is no `WHERE` clause, all rows in the table are returned.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(scans all rows in the `CITY` table)*

**Space Complexity:** `O(1)` *(excluding the output result set)*
