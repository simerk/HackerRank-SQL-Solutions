## 🚀 9. Weather Observation Station 3

### 💡 Idea

We need to retrieve the **names of all cities** from the `STATION` table where the **ID is an even number**.

We can use the modulo (`MOD`) operator to check whether an `ID` is even.

---

### 💻 Solution

```sql
SELECT DISTINCT CITY
FROM STATION
WHERE MOD(ID, 2) = 0;
```

---

### 🛠️ How It Works

1. Start with the **STATION** table.
2. Use `MOD(ID, 2)` to find the remainder when `ID` is divided by `2`.
3. If the remainder is `0`, the `ID` is even.
4. `SELECT DISTINCT` ensures that each city name appears only once, even if multiple stations exist in the same city.

---

### 🔄 Alternative Approaches

**Using the `%` operator** *(supported in some SQL databases)*

```sql
SELECT DISTINCT CITY
FROM STATION
WHERE ID % 2 = 0;
```

This produces the same result but is not supported by every SQL dialect. `MOD()` is more portable.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(scans all rows in the `STATION` table)*

**Space Complexity:** `O(n)` *(for storing distinct city names in the result set)*
